Este projeto é um robô feito com LEGO Mindstorms EV3 capaz de resolver um cubo mágico automaticamente.

Ele utiliza três motores: um para girar o cubo, outro para mudar as faces e um terceiro para movimentar o sensor de cor. O sensor lê todas as cores do cubo e envia essas informações para um computador, que calcula a solução usando um algoritmo específico.

Depois disso, o robô recebe a sequência de movimentos e executa cada passo até resolver o cubo sozinho.
